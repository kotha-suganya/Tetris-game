🧩 Tetris Game

A classic Tetris game implementation where players arrange falling blocks to complete horizontal lines and score points. The game challenges players with increasing speed and requires quick thinking, coordination, and strategic placement of blocks.

This project recreates the nostalgic arcade experience while demonstrating core programming concepts such as game loops, collision detection, and grid-based movement.


🏆 Objective

Arrange falling blocks to complete full horizontal lines.
When a line is completed, it disappears and you earn points.
The game ends when the blocks stack up to the top of the grid.
